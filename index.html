Work Experience

Project Associate Robert Bosch Centre for Data Science and AI, Indian Institute of Tehcnology, Madras, India (December 2020 - July 2021)

I worked under the guidance of Dr. Gitakrishnan Ramadurai and Dr. Balaraman Ravindran in the RBCDSAI, a funded research centre at Indian Institute of Technology, Madras at Chennai, India. I was a part of the team working with thr Tamil Nadu Health Systems Project to develop efficient algorithms to imporvise the Emergency Medicine System (EMS) in Tamil Nadu. I, along with the team, developed a fully interactive dashboard to present real time statistics of the EMS data. This was developed using Dash, Plotly and Holium packages. The dashboard had a comprehensive view of the number of calls arriving, number of ambulances in service, and an interactive geospatial map showing the locations of the bases of ambulances. Using the textual data injested by the EMS personnel at the centre, we developed an algorithm to identify the incident location. We combined a clustering algorithm using DBSCAN and an distance heuristic to identify the incident location. An accuracy of 87% across five different precision levels (location, street, landmark, colony and locality) was observed for this algorithm.
Using the identified incident locations, we designed a static allocation algorithm using optimization techniques with a custom lose function. The aim was to minize the "cost" incurred when an ambulance is allocated to a request. This algorithm optimized the usage of ambulances in the district of Chennai in Tamil Nadu by decreasing it by 13%.

Lead Machine Learning Engineer Omdena (November 2020 - Present)

Omdena is a voluntary organization which specializes in solving real world problems using artificial intelligence. I was an active collaborator in three Omdena projects and I am statrting my fourth in February 2022.

  • Omdena WRI Land Management Project: This project focused on developing a Knowledge Graph for the unstructured data from gazatte and court documents to solve land misappropriations in Kenya. The raw data was collected, annotated, processed to find the important entities and then a Knowledge was constructed using the recognized entities and relationships. I was the task leader for the Named Entity Recognition task and an active participant in the Annotations task. Along with my team, I developed models to detect the major named entities such as places, numbers and owner details using Spacy, BERT and Attention based models. We used a custom set of tags specific to our problem statement such as Owner Name, Deed Number, Property ID etc. We observed that the Spacy model gave the best accuracy of 96% and we extracted the entities from about 36000 gazatte documents from the time period 2005 to 2018.
  • Omdena FINZ Project: This project involved developing a pipeline to include geopolitical and climatic factors while scoring a proposal as feasible or non-feasible. A pipeline which took numeric inputs to identify geopolitical risks and image data to identify climatic risks which then gave a score to the proposal based on the location input was designed. I was an active collaborator in the data collection task where we used image sources such as Sentinal 1, 2 and 2A to collect satellite images for modelling , and the image modelling task were we developed image segmentation models to identify flood prone areas.
  • Omdena GlintSolar Project : This project attempts to develop a solution to the scaricity of land to install solar panels to reduce global warming. The pipeline developed essentially eliminates noise in the the satellite image during the depth estimation process to determine if a piece of land is feasible to install solar pannels. I was an active collaborator in the data collection task where we collected satellite image data to identify the water depth and other affecting factors precisely.
  • Omdena Hashiona Project: This project attempts to find correlations, symptoms and morbidity for Hashimoto’s Disease. I have been selected as a part of this team and this project will commence in February 2022.
